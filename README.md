# 🖥️ ระบบสอบออนไลน์ป้องกันการทุจริต (Anti-Cheat Online Exam WebApp)

ระบบจัดการสอบออนไลน์พัฒนาด้วย **Google Apps Script (GAS)** ร่วมกับ **HTML5/JavaScript** ที่มีความสามารถในการดึงข้อสอบจาก Google Sheets มาแสดงผลตรงบนเว็บเพื่อบล็อกการเข้าถึงแป้นพิมพ์ ป้องกันการคลิกขวา บังคับเข้าโหมดเต็มจอ และตรวจจับการสลับจอ/สลับแท็บได้ 100% พร้อมเก็บข้อมูลคะแนนและประวัติความปลอดภัยใน Google Sheets แบบอัตโนมัติ

---

## 🌟 ฟีเจอร์สำคัญของระบบ

1. **ระบบทำข้อสอบตรงบนเว็บ (Native Exam Engine)**:
   * ดึงข้อสอบและตัวเลือกจากชีตมาเรนเดอร์บนหน้าเว็บโดยตรง ไม่ผ่าน IFrame ป้องกันการโกงได้ดีกว่า
   * **ปลอดภัยสูงสุด**: ระบบดึงข้อมูลโจทย์โดยกรองข้อมูลเฉลยออกก่อนส่งมายังฝั่งนักเรียน เพื่อป้องกันการกดตรวจสอบโค้ด (Inspect) ดูเฉลยหลังบ้าน
2. **ระบบป้องกันการทุจริตอัจฉริยะ (Proctoring & Anti-Cheat)**:
   * **บังคับเข้าโหมดเต็มจอ (Fullscreen Enforcer)**: ต้องทำข้อสอบในโหมดเต็มจอเท่านั้น หากกด Esc หรือยกเลิก จะถือเป็นการละเมิด
   * **ตรวจสลับแท็บและโปรแกรม (Visibility change & Focus loss)**: ตรวจจับเมื่อผู้เรียนย้ายแท็บ ย่อจอ เปิดโปรแกรมอื่น หรือใช้หน้าจอที่สอง (Dual Screen)
   * **บล็อกคีย์บอร์ดและเมาส์ลัด**: ห้ามคลิกขวา, ห้าม F12, ห้าม Ctrl+Shift+I/J/C (DevTools), ห้าม Ctrl+U (ซอร์สโค้ด), ห้าม Ctrl+C/V/X (ก๊อปปี้/วาง/ตัดข้อความ), ห้ามลากเมาส์คลุมดำ, ห้าม Ctrl+S (บันทึกหน้าเว็บ), ห้าม Ctrl+P (สั่งพิมพ์)
   * **ระบบจำกัดความผิด (Violation Limit)**: เมื่อทำผิดกฎความปลอดภัยครบ 2 ครั้ง ระบบจะบันทึกสถานะล็อกและคะแนนปัจจุบันลงระบบทันทีพร้อมปิดกั้นไม่ให้ทำสอบต่อ
3. **ระบบนำเข้าข้อสอบจากไฟล์ Word (`.docx`)**:
   * ครูอัปโหลดไฟล์ Word (`.docx`) ระบบสแกนหาข้อสอบตามรูปแบบ ก. ข. ค. ง. และเฉลยอัตโนมัติ
   * **หน้าต่าง Preview & Edit**: แสดงข้อสอบให้ครูตรวจสอบความถูกต้อง แก้ไขตัวสะกด ปรับเปลี่ยนเฉลย หรือใส่คะแนนก่อนกดบันทึกจริงเข้าชีต
4. **บันทึกคะแนนและประวัติทุจริตอัตโนมัติ**:
   * ชีต `Exam_Results` เก็บประวัติส่งสอบ คะแนนดิบ คะแนนเต็ม และสถานะการส่ง (เช่น ส่งตามปกติ หรือถูกตัดสิทธิ์สอบ)
   * ชีต `Security_Logs` เก็บรวบรวมวินาทีที่เกิดเหตุทุจริต เช่น หลุดเต็มจอ หรือสลับหน้าต่าง

## 🚀 วิธีการติดตั้งบน Google Apps Script

1. เปิดไฟล์ Google Sheets ของคุณ 
2. ไปที่เมนู **ส่วนขยาย (Extensions)** -> **Apps Script**
3. ลบโค้ดเดิมทั้งหมดในไฟล์ `รหัส.gs` (หรือ `Code.gs`) จากนั้นคัดลอกโค้ดใน [Code.js](file:///d:/ระบบสอบออนไลน์%20kss/Code.js) ไปวางแทนที่
4. กดปุ่มเครื่องหมายบวก `+` ข้างๆ เมนูไฟล์ซ้ายมือ เลือกสร้างไฟล์ **HTML** ตั้งชื่อไฟล์ว่า `Index` (ไม่ต้องมีนามสกุล .html)
5. คัดลอกโค้ดจาก [Index.html](file:///d:/ระบบสอบออนไลน์%20kss/Index.html) ไปวางแทนที่ข้อมูลทั้งหมดในไฟล์ `Index.html` ที่สร้างใหม่
6. ตรวจสอบให้แน่ใจว่าค่า `SPREADSHEET_ID` ที่บรรทัดแรกในไฟล์ `Code.js` ตรงกับไอดีชีตของคุณจริง
7. กด **บันทึกโครงการ** (รูปแผ่นดิสก์)
8. คลิกปุ่ม **การทำให้ใช้งานได้ (Deploy)** -> **การทำให้ใช้งานได้ใหม่ (New deployment)**
   * เลือกประเภทเป็น **เว็บแอป (Web app)**
   * กำหนดค่าผู้มีสิทธิ์เรียกใช้งาน: **เป็นฉัน (Me)**
   * กำหนดค่าผู้เข้าถึงเว็บแอป: **ทุกคน (Anyone)**
   * คลิก **ทำให้ใช้งานได้ (Deploy)** และทำการอนุมัติสิทธิ์การเข้าถึงข้อมูล Sheets ให้เรียบร้อย
9. คัดลอก URL เว็บแอปที่ได้รับไปแจกจ่ายให้นักเรียนใช้งานสอบ

## 🛠️ วิธีการนำโค้ดในโฟลเดอร์นี้ขึ้น GitHub

โฟลเดอร์นี้ได้รับการเริ่มต้นระบบ Git เอาไว้แล้ว คุณสามารถนำโปรเจกต์นี้ส่งขึ้น GitHub ของคุณได้ด้วยคำสั่งด้านล่างนี้ (ผ่าน Git Bash หรือ PowerShell):

1. **ตรวจสอบความพร้อมของไฟล์**:
   ```bash
   git status
   ```

2. **เพิ่มไฟล์ทั้งหมดเข้าคิวคอมมิต**:
   ```bash
   git add .
   ```

3. **ทำการคอมมิตประวัติการแก้ไข**:
   ```bash
   git commit -m "Initial commit - Upgrade Anti-Cheat Online Exam with native engine and docx parser"
   ```

4. **เชื่อมต่อและส่งโค้ดขึ้น GitHub**:
   สร้าง Repository เปล่าในหน้าเว็บ GitHub.com จากนั้นรันคำสั่งเหล่านี้ (แทนที่ `<YOUR_GITHUB_REPO_URL>` ด้วยลิงก์ปลายทางของคุณ):
   ```bash
   git branch -M main
   git remote add origin <YOUR_GITHUB_REPO_URL>
   git push -u origin main
   ```
